This directory holds Ada ports of SDL programs that originate from SDL's
upstream `test/` tree instead of its `examples/` tree.

They still belong under `examples/` in this repository because they are useful
binding exercises and manual validation programs, but they are grouped here so
they do not look like uncategorized ports.

Current contents:

- `testbounds`, ported from `SDL/test/testbounds.c`
- `testdisplayinfo`, ported from `SDL/test/testdisplayinfo.c`
- `testfilesystem`, ported from `SDL/test/testfilesystem.c`
- `testlocale`, ported from `SDL/test/testlocale.c`
- `testtime`, ported from `SDL/test/testtime.c`
- `testurl`, ported from `SDL/test/testurl.c`
